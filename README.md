# Skilify

Uma landing page moderna para uma plataforma de aprendizado online.

O **Skilify** é um projeto de estudo focado na criação da **interface inicial de um site educacional**, simulando uma plataforma onde usuários poderiam aprender novas habilidades através de cursos online.

Este projeto foi desenvolvido para praticar **desenvolvimento front-end**, principalmente estruturação de páginas e organização visual.

---

## Preview

A interface simula a **página inicial de uma plataforma de aprendizado**, contendo seções comuns encontradas em plataformas educacionais, como:

- Apresentação da plataforma
- Destaque de cursos
- Mensagem dos alunos
- Chamadas para ação

> ⚠️ Atualmente o projeto contém apenas a **landing page**.

---

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando tecnologias básicas da web:

- **HTML5** — Estrutura da página  
- **CSS3** — Estilização e layout  

Nenhuma biblioteca ou framework foi utilizado, pois o foco foi praticar os fundamentos.

---

## Objetivo do projeto

O projeto foi criado com o objetivo de:

- Praticar **HTML semântico**
- Melhorar habilidades em **CSS**
- Criar layouts organizados
- Simular um projeto real de **landing page**
- Desenvolver projetos para **portfólio**
